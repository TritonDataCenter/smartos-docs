SmartOS Documentation : Using Ansible
This page last changed on Oct 15, 2017 by cjr.

Ansible 2.3 has two SmartOS specific modules:

  • imgadm
  • vmadm

To run these you need python 2 in the global zone. This doesn't exist on SmartOS, so an easy way is to use pkgsrc to install it. See the instructions on the "64-bit tools (SmartOS GZ)" tab at pkgsrc.joyent.com for doing this.

Once that is done, you may write plays using the 2 modules. There are examples in the module documentation and on the module author's blog at:

  • Running Ansible in the SmartOS global zone
  • Ansible modules for SmartOS imgadm and vmadm

Thanks to Jasper Lievisse Adriaanse for writing and sharing these modules!
